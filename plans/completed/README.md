# Completed ExecPlans

Store closed ExecPlans in this directory for historical traceability.

## What belongs here
- Completed (`Complete`) task plans.
- Partially delivered (`Partial`) task plans.
- Cancelled or superseded plans (with status note near the top of the plan).

## Rules
- Keep original filename format: `YYYY-MM-DD_<short-task-name>_execplan.md`.
- Do not delete closed plans unless required by an explicit retention policy.
- Ensure related Completion Reports reference the final plan path.
