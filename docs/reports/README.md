## Major Report Artifacts and Purpose
- `completion_report_template.md`: required handoff template for task completion reporting.
- `*_full_repository_state_audit.md`: broad repository-state findings and remediation priorities.
- `*_consolidated_audit_verification.md`: verification pass confirming/qualifying audit findings.
- `*_completion_report.md`: task-level closeout with objective status, validation evidence, and follow-up notes.

## Maintenance Rule
When adding a new audit, verification, or completion report, update this index in the same change set.
